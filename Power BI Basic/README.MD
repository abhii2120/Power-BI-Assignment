#  Power BI Basic Assignment

This repository contains my **Power BI Basic Assignment**, completed as part of my **Data Analytics coursework**.  
The assignment focuses on understanding the fundamentals of Power BI and implementing them practically using Power BI Desktop.

---

##  Assignment Overview
The objective of this assignment is to gain basic knowledge of Power BI and its components, understand the Power BI workflow, and perform basic data cleaning and transformation using Power Query.

All questions from **Q1 to Q8** have been covered as per the assignment requirements.

---

##  Concepts Covered
The following concepts are covered in this assignment:

- Introduction to Power BI  
- Components of Power BI (Power BI Desktop, Service, and Mobile)  
- Power BI Workflow  
- Data Cleaning using Power Query  
- Loading dataset in Power BI  
- Data View, Report View, and Model View  
- Data Sources supported by Power BI  
- Splitting columns using Power Query
