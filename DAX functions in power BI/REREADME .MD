# DAX Functions in Power BI – Assignment

##  Overview
This repository contains an academic assignment based on **DAX (Data Analysis Expressions)** in **Microsoft Power BI**.  
The assignment demonstrates the use of calculated columns, measures, ranking, filtering, and time intelligence functions to analyze sales data and generate meaningful business insights.

---

##  Files Included
-  DAX function in Power BI (Solved Assignment – PDF)
-  Power BI file (.pbix) containing DAX implementations
-  Question Paper (if applicable)

---

##  Topics Covered
- Calculated Columns in Power BI  
- Measures using DAX  
- IF and Logical Functions  
- Aggregation Functions  
- Time Intelligence (Year-to-Date Sales)  
- Ranking using RANKX  
- Filtering using CALCULATE  
- Business-oriented data analysis using DAX  

---

##  Tools & Technologies Used
- Microsoft Power BI Desktop  
- DAX (Data Analysis Expressions)

---

##  Dataset Used
**Amazon Store Sales Dataset**  
(Number of Records: 500)

---

##  Objective of the Assignment
The objective of this assignment is to understand and apply DAX functions in Power BI for performing data analysis, improving reporting accuracy, and supporting data-driven business decision-making.
