# Publishing & Sharing Reports in Power BI

##  Project Overview
This project focuses on **Publishing and Sharing Power BI reports** using Power BI Service.  
It explains how Power BI reports are published from Power BI Desktop to the Power BI Service and shared securely with users for collaboration and decision-making.

The assignment highlights best practices for report publishing, sharing options, access control, and collaboration features in Power BI.

##  Publishing Reports in Power BI
Publishing involves uploading a Power BI report from Power BI Desktop to the Power BI Service.  
Once published, the report becomes available online and can be accessed through a web browser.

---

##  Sharing Reports in Power BI
Power BI allows reports to be shared with specific users or groups.  
Access can be controlled by assigning permissions such as:
- View access
- Edit access

This ensures data security while enabling collaboration.
