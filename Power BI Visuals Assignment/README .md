# Power BI Visuals Assignment

##  Overview
This repository contains an academic assignment focused on **Data Visualization using Microsoft Power BI**.  
The assignment demonstrates the use of different Power BI visuals to represent data clearly, effectively, and meaningfully for business analysis.

---

##  Files Included
-  POWER BI VISUALS ASSIGNMENT.pbix – Power BI report file
-  POWER BI VISUALS ASSIGNMENT.pdf – Solved assignment (exported)
-  POWER BI Visuals Question Paper.pdf – Assignment questions
-  README.md – Project description

---

##  Visuals Covered
- Bar Chart  
- Column Chart  
- Line Chart  
- Pie Chart  
- Donut Chart  
- Table and Matrix  
- Cards and KPIs  
- Slicers and Filters  

---

##  Tools Used
- Microsoft Power BI Desktop

---

##  Objective
The objective of this assignment is to understand how different visuals in Power BI can be used to analyze data, highlight trends, compare values, and support data-driven decision making.
