# Advanced Visuals in Power BI – Assignment

##  Project Overview
This repository contains the **Advanced Visuals in Power BI assignment** created as part of the Power BI coursework.  
The project focuses on implementing advanced Power BI features such as filters, slicers, drill-down, drill-through, and interactive dashboards using a sales dataset.

---

##  Files Included

- **Advanced Visuals in Power BI theory.pbix**  
  Power BI report file containing implemented visuals, filters, slicers, drill-down and drill-through functionality.

- **Advanced Visuals in Power BI theory.pdf**  
  PDF file containing detailed **theoretical explanations** of all assignment questions.

- **Advanced visual Question paper.pdf**  
  Original assignment question paper provided for reference.

- **README.md**  
  Project documentation and overview.

---

##  Tools & Technologies Used
- Microsoft Power BI Desktop  
- DAX (Data Analysis Expressions)  
- GitHub  

---

##  Key Concepts Covered
- Basic Filter and Advanced Filter  
- Visual-Level, Page-Level, and Report-Level Filters  
- Slicers for interactive filtering  
- Drill-Down using hierarchies  
- Drill-Through for detailed analysis  
- Date hierarchy analysis (Year → Quarter → Month → Day)  
- Measure-based filtering using DAX  
- Interactive dashboard design  

---

##  Learning Outcome
Through this assignment, I learned how to:
- Build interactive and user-friendly Power BI dashboards  
- Apply different levels of filters effectively  
- Use drill-down and drill-through for deep data analysis  
- Create and apply DAX measures for business insights
